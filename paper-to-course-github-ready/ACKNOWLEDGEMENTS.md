# Acknowledgements

This project was inspired by Zara Zhang's Codebase-to-Course:

https://github.com/zarazhangrui/codebase-to-course

The original project demonstrated a strong interaction pattern for turning codebases into readable, visual, browser-based courses. This repository explores a related but different direction: turning research papers into interactive study courses.

## What Was Adapted

The current `paper-to-course` prototype adapts the general course format and interaction style:

- scroll-based module navigation
- progress bar and navigation dots
- translation blocks
- quiz interactions
- flow animations
- tooltip glossary interactions
- card/callout visual language

## What Is New Here

The paper-reading workflow and TimeGrad course content are newly created:

- PDF/paper-oriented analysis workflow
- problem -> Gap -> claim -> method -> equations -> experiments -> limits -> reuse structure
- equation and algorithm teaching prompts
- figure/table reading prompts
- experiment audit framing
- reproducibility checklist framing
- Chinese TimeGrad course text and quizzes

If you reuse this repository publicly, please keep this acknowledgement visible.
