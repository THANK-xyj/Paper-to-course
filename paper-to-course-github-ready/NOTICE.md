# Notice

This repository contains a prototype derived from multiple sources of inspiration and content.

## Interaction Template Attribution

The interactive course format in `docs/` and `skills/paper-to-course/assets/` is adapted from the interaction style of Zara Zhang's Codebase-to-Course:

https://github.com/zarazhangrui/codebase-to-course

Please check the original repository before reusing those assets in another public or commercial project.

## Paper Figure Attribution

The demo course uses figures and tables extracted from the TimeGrad paper:

> Kashif Rasul, Calvin Seward, Ingmar Schuster, Roland Vollgraf. "Autoregressive Denoising Diffusion Models for Multivariate Probabilistic Time Series Forecasting." Proceedings of the 38th International Conference on Machine Learning, PMLR 139, 2021.

These figures are included for educational demonstration of the paper-to-course workflow. If you are preparing a public release and want to avoid any copyright ambiguity, remove `docs/paper-assets/` or replace the figures with your own diagrams.
