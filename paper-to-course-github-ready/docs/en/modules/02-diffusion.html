The compact equation

EQUATION 
x_n = sqrt(alpha_bar_n) x_0
    + sqrt(1 - alpha_bar_n) epsilon

epsilon_theta(x_n, h, n) -> epsilon
PLAIN ENGLISH

Take the true vector and mix in Gaussian noise according to the diffusion level.

The diffusion level n controls how damaged the vector is.

The model learns to recover the noise while using the RNN state as context.

x_0the clean observed vector at one time step
hthe RNN hidden state summarizing history
nthe current diffusion noise level
